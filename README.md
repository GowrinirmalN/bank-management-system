# Bank Management System

Python Tkinter Banking Application with SQLite database.

## Features
- User Registration
- User Login
- Create Account
- Check Balance
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- View Account Details
- Close Account
- Professional Dashboard UI

## Technologies Used
- Python
- Tkinter
- SQLite3
